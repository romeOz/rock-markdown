Markdown parser for PHP
=================

Abstraction over the [cebe/markdown](https://github.com/cebe/markdown)

Features
-------------------

 * Deny tags
 * Video tag + dummy:
    * youtube
    * vimeo
    * rutube
    * VK
    * ivi
    * dailymotion
    * sapo
 * Thumb image **(option)**
 * Module for [Rock Framework](https://github.com/romeOz/rock)
 
Requirements
-------------------
 * **PHP 5.4+**
